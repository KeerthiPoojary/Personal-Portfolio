# Keerthi Bangera - Personal Portfolio Website

## Project Objective
Build a personal portfolio website to showcase skills, projects, and resume as a web developer/UI designer.

## Website Structure
- **Home**: Hero section with introduction and typing animation.
- **About**: Bio, education, skills, and interests.
- **Projects**: Dynamically generated project cards (interactive).
- **Contact**: Functional contact form with validation, plus social links.

## Design Decisions
- **Color Scheme**: Navy blue (`#4a6fa5`) as primary for trust & creativity; light gray background for contrast.
- **Typography**: Clean sans-serif (Segoe UI) for readability.
- **Layout**: Bootstrap grid + custom Flexbox for responsiveness.
- **Interactive Elements**: Typing effect, hover animations on cards/buttons, form validation.

## Tools & Platforms
- **Front-end**: HTML5, CSS3, JavaScript (ES6), Bootstrap 5
- **IDE**: VS Code
- **Icons**: Font Awesome
- **Deployment**: Compatible with Netlify / GitHub Pages

## Responsiveness
- Bootstrap breakpoints (sm, md, lg)
- Custom media queries for mobile (max-width: 768px)
- Flexbox for card alignment

## Performance & SEO
- Meta tags for description, keywords, author.
- Semantic HTML (`section`, `nav`, `footer`).
- Lightweight images (placeholders used; replace with optimized images).
- Minify CSS/JS for production.

## How to Run Locally
1. Download all files (`index.html`, `style.css`, `script.js`).
2. Keep them in the same folder.
3. Open `index.html` in any modern browser.

## Deployment Instructions (Netlify/GitHub Pages)
- **GitHub Pages**: Push repo to GitHub → Settings → Pages → Branch `main` → Save.
- **Netlify**: Drag & drop folder to Netlify Drop or connect GitHub.
